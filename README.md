# NFL Concusions EDA and Predictive Model

This project cleans, analyses, and visualizes NFL concussion data from 2012 to 2014; differentiating head injuries and concussions, finding relevant trend lines, contributing factors. And then using multiple regressor models (Linear, Random Forrest, Gradient Boosting, Ada Boost) to predict number of games missed.
